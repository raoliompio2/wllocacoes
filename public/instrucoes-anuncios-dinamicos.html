Instruções para Anúncios Dinâmicos do Google

Nota: Este documento fornece instruções detalhadas para configurar anúncios dinâmicos de pesquisa no Google Ads para o site WL Locações.

1. Feeds de Produtos Disponíveis

Fornecemos três formatos de feeds de produtos que podem ser usados no Google Ads:

Formato URL Uso Recomendado
XML/RSS https://wllocacoes.com.br/api/product-feed?token=wllocacoes-feed-token&format=xml Configuração principal de anúncios dinâmicos
CSV https://wllocacoes.com.br/api/product-feed?token=wllocacoes-feed-token&format=csv Importação para planilhas ou outros sistemas
JSON https://wllocacoes.com.br/api/product-feed?token=wllocacoes-feed-token&format=json Integração com aplicativos personalizados

2. Configurando Anúncios Dinâmicos no Google Ads

Passo 1: Criar uma Nova Campanha

  1. Faça login na sua conta Google Ads
  2. Clique em "Campanhas" e depois em "+ Nova campanha"
  3. Selecione "Gerar vendas", "Gerar leads" ou "Tráfego para o site" como objetivo da campanha
  4. Selecione "Pesquisa" como tipo de campanha
  5. Marque "Anúncios dinâmicos de pesquisa" como subtipo da campanha
  6. Configure as outras opções da campanha (orçamento, ofertas, segmentação geográfica, etc.)

Passo 2: Configurar o Feed de Produtos

  1. Na seção "Configurações dinâmicas", selecione "Use um feed"
  2. Clique em "Novo feed"
  3. Insira as seguintes informações:
    • Nome do feed: "Feed WL Locações"
    • Plano de atualização: "Diariamente" ou "Semanalmente"
    • Tipo de feed: "URL do feed"
    • URL do feed: https://wllocacoes.com.br/api/product-feed?token=wllocacoes-feed-token&format=xml
    • Formato do feed: "XML"
  4. Clique em "Salvar e continuar"
Importante: Certifique-se de que a URL do feed esteja correta e que o token de acesso seja válido.

Passo 3: Configurar Segmentação de Anúncios

  1. Na seção "Segmentações de anúncios dinâmicos", selecione "Usar categorias recomendadas pelo Google"
  2. Opcional: Adicione segmentações adicionais como "Categorias personalizadas" baseadas no atributo custom_label_0 do feed

Passo 4: Criar Grupo de Anúncios e Anúncios

  1. Crie um novo grupo de anúncios
  2. Adicione um anúncio dinâmico de pesquisa no formato que preferir
  3. Configure as descrições do anúncio
  4. Salve o anúncio

Passo 5: Otimizações Adicionais

  1. Configure extensões de anúncios (sitelinks, callouts, estruturados, etc.)
  2. Configure ajustes de lance por dispositivo, localização ou horário
  3. Defina exclusões de pesquisa para termos irrelevantes

3. Categorias Disponíveis para Segmentação

As seguintes categorias de equipamentos estão disponíveis no feed e podem ser usadas para segmentação:

  • Escavadeiras
  • Betoneiras
  • Carregadeiras
  • Andaimes
  • Compactadores
  • Geradores
  • E outras categorias conforme disponíveis no site

4. Manutenção e Atualização

O feed de produtos é atualizado automaticamente quando novos produtos são adicionados, modificados ou removidos do site. Algumas dicas para manutenção:

  • Verifique regularmente se o Google Ads está conseguindo acessar o feed sem erros
  • Monitore o desempenho dos anúncios dinâmicos por categoria
  • Ajuste as descrições dos anúncios conforme necessário para melhorar o CTR
  • Adicione palavras-chave negativas para evitar exibições em pesquisas irrelevantes

Atualização do Feed: O feed é atualizado diariamente às 3:00 da manhã. Quaisquer alterações nos produtos serão refletidas no feed após esse horário.

5. Solução de Problemas

O feed não está sendo importado corretamente

  • Verifique se a URL do feed está correta
  • Confirme se o token de acesso é válido
  • Verifique se o Google Ads tem permissão para acessar o feed

Os anúncios não estão sendo exibidos

  • Certifique-se de que o site esteja indexado corretamente pelo Google
  • Verifique se as segmentações de anúncios estão configuradas corretamente
  • Confirme se o orçamento diário é suficiente

Categorias não estão sendo reconhecidas

  • Verifique se as categorias no feed correspondem às categorias segmentadas no Google Ads
  • Certifique-se de que os produtos têm categorias atribuídas corretamente no sistema
